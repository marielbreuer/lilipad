# Community Librarian Programme — Interactive Activities

A prototype set of 10 browser-based interactive learning activities for the Lilipad Community Librarian Programme.

**Try it here:** [https://marielbreuer.github.io/lilipad/librarian-activities-all.html](https://marielbreuer.github.io/lilipad/librarian-activities-all.html)

---

## What's included

A single HTML file with 10 self-contained activities, all themed around the community librarian role:

1. **Sort It Out** — drag & drop phrases into IS / IS NOT columns
2. **What Would You Do?** — branching patron scenarios with feedback
3. **Key Concepts** — flip cards with definitions
4. **True or False?** — statements with explanations
5. **Explore the Library** — clickable illustrated library map
6. **Reflection Journal** — guided writing prompt
7. **Module Confidence Checklist** — self-assessment with progress bar
8. **Put It in Order** — drag-to-sequence planning steps
9. **How Confident Am I?** — skill confidence sliders
10. **Memory Match** — term and definition matching game

## About

This prototype was built as part of the Lilipad e.V. Community Librarian Programme learning design work. It is intended to demonstrate interactive activity formats for potential use in a self-paced online course.

No frameworks or dependencies — runs entirely in the browser as a single `.html` file.
